Fantasy Football Draft Tracker - Setup Guide

Overview

This guide will help you create a Google Sheets fantasy draft tracker that automatically:

  • Organizes teams across columns with team names in row 1
  • Lists required positions down column A in your specified order
  • Tracks drafted players and automatically assigns them to appropriate roster spots
  • Fills starters first (natural position → superflex → flex), then bench spots
  • Provides position counts and draft analysis

Key Formulas

Position Requirements Setup (Column A)

In A2 and down, manually enter your position sequence:

QB, RB, RB, WR, WR, TE, FLEX, SUPERFLEX, BN1, BN2, BN3...
Main Roster Formula (for each team column, starting at B2)

This master formula handles all the auto-fill logic:

=IF(ROW()<=9, IF($A2="QB", IF(COUNTIF(B$2:B2,$A2)COUNTIF($A$2:$A$9,"RB"), INDEX('Draft Picks'!$D:$D,SMALL(IF(('Draft Picks'!$C:$C=B$1)*('Draft Picks'!$E:$E="RB"),ROW('Draft Picks'!$D:$D)),COUNTIF($A$2:$A$9,"RB")+1)), IF(COUNTIFS('Draft Picks'!$C:$C,B$1,'Draft Picks'!$E:$E,"WR")>COUNTIF($A$2:$A$9,"WR"), INDEX('Draft Picks'!$D:$D,SMALL(IF(('Draft Picks'!$C:$C=B$1)*('Draft Picks'!$E:$E="WR"),ROW('Draft Picks'!$D:$D)),COUNTIF($A$2:$A$9,"WR")+1)), IF(COUNTIFS('Draft Picks'!$C:$C,B$1,'Draft Picks'!$E:$E,"TE")>COUNTIF($A$2:$A$9,"TE"), INDEX('Draft Picks'!$D:$D,SMALL(IF(('Draft Picks'!$C:$C=B$1)*('Draft Picks'!$E:$E="TE"),ROW('Draft Picks'!$D:$D)),COUNTIF($A$2:$A$9,"TE")+1)), ""))), IF($A2="SUPERFLEX", IF(COUNTIFS('Draft Picks'!$C:$C,B$1,'Draft Picks'!$E:$E,"QB")>COUNTIF($A$2:$A$9,"QB"), INDEX('Draft Picks'!$D:$D,SMALL(IF(('Draft Picks'!$C:$C=B$1)*('Draft Picks'!$E:$E="QB"),ROW('Draft Picks'!$D:$D)),COUNTIF($A$2:$A$9,"QB")+1)), ""), ""))))), INDEX('Draft Picks'!$D:$D,SMALL(IF(('Draft Picks'!$C:$C=B$1)*(COUNTIF(B$2:B$9,'Draft Picks'!$D:$D)=0),ROW('Draft Picks'!$D:$D)),ROW()-10)))
Simplified Bench Formula (for rows 11+)
=IFERROR(INDEX('Draft Picks'!$D:$D,SMALL(IF(('Draft Picks'!$C:$C=B$1)*(COUNTIF(B$2:B$10,'Draft Picks'!$D:$D)=0),ROW('Draft Picks'!$D:$D)),ROW()-10)),"")

Position Counter Formulas

Count by Position (create a summary section)

QB Count for Team 1:

=COUNTIFS('Draft Picks'!$C:$C,"Team 1",'Draft Picks'!$E:$E,"QB")

RB Count for Team 1:

=COUNTIFS('Draft Picks'!$C:$C,"Team 1",'Draft Picks'!$E:$E,"RB")

Setup Instructions

  1. Create Sheet1 (Main Draft Sheet):
    • Put "Position" in A1
    • Enter team names in B1, C1, D1, etc.
    • List positions in column A starting from A2 (QB, RB, RB, WR, WR, TE, FLEX, SUPERFLEX, BN1, BN2...)
  2. Create Sheet2 (Draft Picks):
    • Headers: Pick # | Round | Team | Player | Position
    • Enter picks as they happen during the draft
  3. Apply the Main Formula:
    • Copy the main roster formula to B2
    • Drag down to fill all roster spots
    • Copy across to all team columns

How It Works

  • Natural Position First: Players fill their drafted position spots first
  • Superflex Priority: Extra QBs go to superflex before bench
  • Flex Priority: Extra RB/WR/TE go to flex before bench
  • Bench Filling: Remaining players fill bench spots in draft order
  • Real-time Updates: As you enter picks in the Draft Picks sheet, rosters update automatically

Pro Tips:

  • Use conditional formatting to highlight filled positions
  • Add a "Position Needs" section that shows which positions each team still needs
  • Create a "Best Available" sheet that filters out already drafted players
  • Use data validation on the Draft Picks sheet to ensure position accuracy
